Cisco 2960 交换机是企业级入门级交换机，常用命令涵盖 **基础配置、接口管理、VLAN 配置、安全管理、监控调试** 等核心场景。以下是按功能分类的 **常用命令大全**，附带简洁说明和使用场景，零基础也能直接套用：


## 一、基础模式与全局配置命令
### 1. 模式切换命令
| 命令                  | 功能描述                                  | 适用模式       |
|-----------------------|-------------------------------------------|----------------|
| `enable`（简写 `en`） | 从用户模式进入特权模式                    | 用户模式（`Switch>`） |
| `disable`             | 从特权模式退回用户模式                    | 特权模式（`Switch#`） |
| `configure terminal`（简写 `config t`） | 进入全局配置模式                | 特权模式       |
| `exit`                | 退回上一级模式（如接口→全局、全局→特权）  | 任意非用户模式 |
| `end` / `Ctrl+Z`      | 从任意模式直接返回特权模式                | 任意模式       |
| `?`                   | 查看当前模式下所有可用命令（如 `conf?`）  | 任意模式       |
| `Tab`                 | 命令自动补全（如输入 `int` 按 Tab 补全）  | 任意模式       |

### 2. 全局基础配置
| 命令                                  | 功能描述                                  | 示例                                  |
|---------------------------------------|-------------------------------------------|---------------------------------------|
| `hostname [名称]`                     | 修改交换机主机名（便于区分设备）          | `Switch(config)# hostname SW-Office`   |
| `no ip domain lookup`                 | 关闭域名解析（避免输错命令时卡顿）        | `SW-Office(config)# no ip domain lookup` |
| `banner motd # [提示语] #`             | 设置登录横幅（如版权提示、安全警告）      | `SW-Office(config)# banner motd # 禁止未授权登录 #` |
| `enable password [明文密码]`           | 设置特权模式明文密码（不推荐，易泄露）    | `SW-Office(config)# enable password cisco123` |
| `enable secret [加密密码]`             | 设置特权模式加密密码（优先级高于明文密码）| `SW-Office(config)# enable secret cisco@2024` |
| `service password-encryption`          | 加密所有明文密码（如Console、Telnet密码） | `SW-Office(config)# service password-encryption` |


## 二、接口配置命令（物理接口/逻辑接口）
### 1. 接口基础配置
| 命令                                  | 功能描述                                  | 示例                                  |
|---------------------------------------|-------------------------------------------|---------------------------------------|
| `interface [接口类型] [编号]`（简写 `int`） | 进入接口配置模式（物理口/ VLAN 口）      | 物理口：`int fa 0/1`（FastEthernet 0/1）<br>VLAN 口：`int vlan 1` |
| `no shutdown`（简写 `no shut`）        | 启用接口（默认关闭，必须执行）            | `SW-Office(config-if)# no shut`       |
| `shutdown`（简写 `shut`）              | 禁用接口（接口故障时临时关闭）            | `SW-Office(config-if)# shut`          |
| `speed [10/100/1000/auto]`            | 设置接口速率（2960 支持百兆/千兆自适应）  | `SW-Office(config-if)# speed 100`     |
| `duplex [half/full/auto]`             | 设置双工模式（全双工/半双工/自适应）      | `SW-Office(config-if)# duplex full`   |
| `description [描述]`                  | 给接口添加备注（如连接设备、用途）        | `SW-Office(config-if)# description Connect-PC-01` |

### 2. 批量接口配置（效率优化）
| 命令                                  | 功能描述                                  | 示例                                  |
|---------------------------------------|-------------------------------------------|---------------------------------------|
| `interface range [接口范围]`           | 批量进入多个接口（避免重复操作）          | 连续口：`int range fa 0/1-10`（0/1到0/10）<br>离散口：`int range fa 0/1,fa 0/5` |
| `switchport mode access`              | 批量将接口设为访问模式（连接终端设备）    | `SW-Office(config-if-range)# switchport mode access` |
| `switchport access vlan [VLAN号]`     | 批量将接口划分到指定VLAN                  | `SW-Office(config-if-range)# switchport access vlan 10` |

### 3. VLAN 接口配置（远程管理）
| 命令                                  | 功能描述                                  | 示例                                  |
|---------------------------------------|-------------------------------------------|---------------------------------------|
| `interface vlan [VLAN号]`              | 进入VLAN逻辑接口（配置IP用于远程管理）    | `SW-Office(config)# int vlan 1`       |
| `ip address [IP地址] [子网掩码]`       | 为VLAN接口配置IP（必须与终端同网段）      | `SW-Office(config-if)# ip address 192.168.1.253 255.255.255.0` |
| `ip default-gateway [网关IP]`          | 设置默认网关（跨网段远程管理时需配置）    | `SW-Office(config)# ip default-gateway 192.168.1.1` |


## 三、VLAN 与 trunk 配置命令
### 1. VLAN 基础配置
| 命令                                  | 功能描述                                  | 示例                                  |
|---------------------------------------|-------------------------------------------|---------------------------------------|
| `vlan [VLAN号]`                       | 创建VLAN并进入VLAN配置模式（1-4094）      | `SW-Office(config)# vlan 10`          |
| `name [VLAN名称]`                     | 为VLAN命名（便于识别用途）                | `SW-Office(config-vlan)# name Office-PC` |
| `no vlan [VLAN号]`                    | 删除指定VLAN（需先将接口移出该VLAN）      | `SW-Office(config)# no vlan 10`       |
| `switchport mode trunk`               | 将接口设为 trunk 模式（连接其他交换机）    | `SW-Office(config-if)# switchport mode trunk` |
| `switchport trunk allowed vlan [VLAN范围]` | 限制 trunk 允许通过的VLAN（默认允许所有） | 允许指定VLAN：`switchport trunk allowed vlan 10,20` <br>允许所有：`switchport trunk allowed vlan all` |


## 四、安全配置命令（防攻击/权限控制）
### 1. 端口安全（防止非法设备接入）
| 命令                                  | 功能描述                                  | 示例                                  |
|---------------------------------------|-------------------------------------------|---------------------------------------|
| `switchport port-security`            | 启用端口安全功能（默认关闭）              | `SW-Office(config-if)# switchport port-security` |
| `switchport port-security maximum [数量]` | 限制端口最大接入设备数（默认1）          | `SW-Office(config-if)# switchport port-security maximum 2` |
| `switchport port-security mac-address sticky` | 自动学习端口连接设备的MAC地址（粘性MAC）  | `SW-Office(config-if)# switchport port-security mac-address sticky` |
| `switchport port-security violation [protect/shutdown/restrict]` | 违规处理（Protect：丢弃数据；Shutdown：关闭端口；Restrict：告警） | `SW-Office(config-if)# switchport port-security violation shutdown` |

### 2. 登录安全（Console/Telnet/SSH）
| 命令                                  | 功能描述                                  | 示例                                  |
|---------------------------------------|-------------------------------------------|---------------------------------------|
| `line console 0`                      | 进入Console控制台线路配置（本地登录）     | `SW-Office(config)# line console 0`    |
| `password [密码]`                     | 设置Console登录密码                       | `SW-Office(config-line)# password console123` |
| `login`                               | 启用Console密码验证（必须配置）           | `SW-Office(config-line)# login`       |
| `logging synchronous`                 | 日志信息与命令输入同步（避免干扰输入）    | `SW-Office(config-line)# logging synchronous` |
| `line vty 0 4`                        | 进入Telnet线路配置（允许5个并发会话）     | `SW-Office(config)# line vty 0 4`     |
| `transport input telnet ssh`           | 允许Telnet和SSH远程登录（推荐仅启用SSH）  | `SW-Office(config-line)# transport input ssh` |


## 五、监控与调试命令（特权模式）
| 命令                                  | 功能描述                                  | 示例                                  |
|---------------------------------------|-------------------------------------------|---------------------------------------|
| `show running-config`（简写 `show run`） | 查看当前生效的配置（内存中的配置）        | `SW-Office# show run`                 |
| `show startup-config`（简写 `show start`） | 查看启动配置（NVRAM中的配置，重启生效）    | `SW-Office# show start`               |
| `show interface [接口名]`              | 查看接口状态（速率、双工、流量、错误包）  | `SW-Office# show interface fa 0/1`    |
| `show vlan brief`                     | 查看VLAN简要信息（VLAN号、名称、所属接口）| `SW-Office# show vlan brief`          |
| `show mac-address-table`              | 查看MAC地址表（设备MAC与接口的映射）      | `SW-Office# show mac-address-table`   |
| `show ip interface brief`（简写 `show ip int brief`） | 查看IP接口状态（IP、掩码、是否启用）      | `SW-Office# show ip int brief`        |
| `show cdp neighbors`（简写 `show cdp neig`） | 查看CDP邻居设备（连接的其他Cisco设备）    | `SW-Office# show cdp neig`            |
| `ping [IP地址]`                       | 测试与目标设备的连通性                    | `SW-Office# ping 192.168.1.1`         |
| `traceroute [IP地址]`（简写 `trace`）  | 追踪数据包传输路径（排查网络不通问题）    | `SW-Office# trace 192.168.2.1`        |


## 六、配置保存与恢复命令（特权模式）
| 命令                                  | 功能描述                                  | 示例                                  |
|---------------------------------------|-------------------------------------------|---------------------------------------|
| `write memory`（简写 `write`）         | 保存当前配置到NVRAM（重启不丢失）         | `SW-Office# write`                    |
| `copy running-config startup-config`（简写 `copy run start`） | 等同于 `write`，保存配置 | `SW-Office# copy run start`           |
| `copy startup-config running-config`（简写 `copy start run`） | 加载启动配置到当前生效配置 | `SW-Office# copy start run`           |
| `erase startup-config`（简写 `erase start`） | 清空启动配置（恢复出厂设置前执行）        | `SW-Office# erase start`              |
| `reload`                              | 重启交换机（配置修改后需重启生效时使用）  | `SW-Office# reload`（按Y确认）        |


## 七、常用命令速查总结（高频场景）
| 应用场景                | 核心命令组合                                                                 |
|-------------------------|------------------------------------------------------------------------------|
| 初始化交换机            | `en → config t → hostname SW1 → no ip domain lookup → enable secret 123 → service password-encryption` |
| 配置VLAN并划分接口      | `config t → vlan 10 → name PC → exit → int range fa 0/1-5 → switchport mode access → switchport access vlan 10 → no shut` |
| 配置远程管理（Telnet）  | `config t → int vlan 1 → ip add 192.168.1.253 255.255.255.0 → no shut → ip default-gateway 192.168.1.1 → line vty 0 4 → password 456 → login` |
| 启用端口安全            | `int fa 0/1 → switchport port-security → switchport port-security mac-address sticky → switchport port-security violation shutdown` |
| 验证配置                | `show vlan brief → show ip int brief → show mac-address-table → ping 192.168.1.1` |
| 保存配置                | `write` 或 `copy run start`                                                  |
